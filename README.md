# untitled friendsim

### dev instructions

edit stuff only in this directory (not in `fansim-engine/`)

to "compile":

- `make clean default`

to run:

- `make renpy`
- preferences -> set "Projects Directory" to `fansim-engine/projects`
- launch the game named "work"
